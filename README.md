# Mathema
Mathema was created to read and solve simple equations

**Example**
(2+1)^2-Cos(0)*-2-1  
gives  
10  

**Understands**
  * Binary operators
    * '+'  add
    * '-'  subtract  
    * '*'  multiply
    * '/'  divide
    * '^'  power
  * Unary operators
    * '-'  sign change
  * Functions
    * 'sin' Sinus
    * 'cos' Cosinus
  * Constants
    * 'Pi' - ratio of a circle's circumference to its diameter
    * 'e' - natural logarithm base
