# Mathema
Mathema was created to read and solve simple equations

**Understands**
  * Binary operators
    * '+'  add
    * '-'  subtract  
    * '*'  multiply
    * '/'  divide
    * '^'  power
  * Unary operators
    * '-'  sign change
  * Functions
    * 'sin' Sinus
    * 'cos' Cosinus
  
 **Plans**
   - variables
   - equations
